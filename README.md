# Sales Performance Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance, customer demographics, product categories, and monthly sales trends. The dashboard helps identify key business insights and supports data-driven decision-making.

## Key Features

* KPI Tracking
* Sales Trend Analysis
* Product Category Analysis
* Customer Demographics Analysis
* Interactive Filters and Slicers
* Monthly Performance Reporting

## Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization
* Interactive Dashboards

## Key Insights

* Total Sales reached ₹456K across 1,000 transactions.
* Electronics generated the highest sales revenue.
* Clothing recorded the highest quantity sold.
* Female customers contributed slightly more purchases than male customers.
* Sales peaked during May and declined in September.
* Customers aged 45–54 represented the largest customer segment.
